# Pokédex Front-End App

![image](https://github.com/user-attachments/assets/530b879d-eafa-4d73-99a3-7f62bee0080c)

## Key Features
- **Search & Filter**: Search Pokémon by name or Pokédex number.
- ![image](https://github.com/user-attachments/assets/3ca95227-3cc1-4278-a568-262266e97d12)
- **Dynamic Type Colors**: Pokémon types are styled with unique colors.
- **Responsive Design**: Fully supports PC and mobile views with a collapsible side navigation menu.
    - Mobile:
      ![image](https://github.com/user-attachments/assets/285cf2cb-0379-406d-adfb-13654b733a10)

      ![image](https://github.com/user-attachments/assets/0cc924c1-2166-4f8b-8e84-3f19d54a606a)

      ![image](https://github.com/user-attachments/assets/65029520-5434-46f2-b395-33d0272158de)

- **Fast Navigation**: Browse through the first 151 Pokémon with ease, with the help of API fetches from [PokéAPI](https://pokeapi.co/) and leveraging cache (local storage)
- **Interactive Interface**: Highlights selected Pokémon and adapts dynamically to user input.
